Pipeline Components
-------------------
The **RIIPL Pipeline** is an SPM-based image processing pipeline that is initiated from RIIPL XNAT.

#. SPM Segmentation
#. VBM8 Segmentation
#. Lesion Segmentation
#. Arterial Spin Labeling
#. PASL
#. PCASL
#. Network Analysis
#. FreeSurfer
